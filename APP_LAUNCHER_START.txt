🔧 WHAT TO DO IF IT DOESN'T WORK
═════════════════════════════════════════════════════════════════════════════════

Problem: App won't start
Solution 1: Check Python installed
  Windows Start → python
  If nothing happens, install Python from python.org

Solution 2: Run in command line to see error
  cd "c:\Users\ytc-y\code projects"
  python launcher.py

────────────────────────────────────────────────────────────────────────────────

Problem: Buttons don't work
Solution: Check console for error messages
  They appear in the output window
  Take a screenshot and check QUICK_START.md

────────────────────────────────────────────────────────────────────────────────

Problem: Server won't start
Solution 1: Click "Stop Server" and try again
Solution 2: Check if port 5000 is in use
  In PowerShell: netstat -ano | findstr :5000
  Kill process if needed: taskkill /PID <pid> /F

────────────────────────────────────────────────────────────────────────────────

Problem: Browser doesn't open
Solution: Click "Open Browser" button again
  Or manually type: http://localhost:5000 in browser
