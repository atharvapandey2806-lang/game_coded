🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

PROBLEM: "Port 5000 already in use"
SOLUTION 1: Kill existing process
  taskkill /IM python.exe /F
  Then run START_SERVER.bat again

SOLUTION 2: Use different port
  Edit gamebot_multiplayer_server.py (last line)
  Change: app.run(port=5000)
  To: app.run(port=5001)
  Then access: http://localhost:5001

────────────────────────────────────────────────────────────────────────────────

PROBLEM: "Module not found: flask"
SOLUTION: Install dependencies
  pip install -r requirements.txt
  Then run START_SERVER.bat

────────────────────────────────────────────────────────────────────────────────

PROBLEM: "Cannot create database"
SOLUTION: Check permissions
  Run Command Prompt as Administrator
  Then run START_SERVER.bat

────────────────────────────────────────────────────────────────────────────────

PROBLEM: Browser shows "Connection refused"
SOLUTION 1: Is server running?
  Check console for "Running on http://0.0.0.0:5000"
  
SOLUTION 2: Try different URL
  http://127.0.0.1:5000 (instead of localhost)
  
SOLUTION 3: Firewall
  Windows Firewall might block Python
  Allow Python through firewall:
  Windows Defender → Firewall → Allow app → Python

────────────────────────────────────────────────────────────────────────────────

PROBLEM: "bcrypt module not found"
SOLUTION:
  pip install bcrypt
  Then restart server

────────────────────────────────────────────────────────────────────────────────

PROBLEM: JavaScript errors in console
SOLUTION: Clear browser cache
  Chrome: Ctrl + Shift + Delete
  Edge: Ctrl + Shift + Delete
  Firefox: Ctrl + Shift + Delete
  Then hard refresh: Ctrl + F5
