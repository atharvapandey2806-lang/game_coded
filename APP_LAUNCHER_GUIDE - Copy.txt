Solution: tkinter is built-in to Python 3.8+

  Click Stop, close any other Python apps, click Start again

  $lnk = $shell.CreateShortcut("$env:USERPROFILE\Desktop\GameBot.lnk")

  $lnk.WorkingDirectory = "c:\Users\ytc-y\code projects"
  $lnk.IconLocation = "python.exe"
  $lnk.Save()

🌐 WHAT HAPPENS BEHIND THE SCENES
════════════════════════════════════════════════════════════════════════════════

When you click "Start Server":

1. launcher.py checks if Python requirements are installed
2. Installs any missing packages (Flask, etc.)
3. Launches gamebot_multiplayer_server.py in background
4. Captures server output in real-time
5. Displays it in the console window
6. Updates status indicator

When you click "Open Browser":

1. Waits 1 second for server to stabilize
2. Opens your default browser
3. Navigates to http://localhost:5000
4. master.html loads and displays the interface
